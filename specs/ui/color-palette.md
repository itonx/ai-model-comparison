# Dark theme

- Background: #202833
- Accent: #24e4a7
- Surface/card: #2a3442
- Border: #344154
- Muted text: #9fb0c2

## Notes

### Add “accent glow” for dark theme

Use this glow only on:

- active tool
- focused search input
- primary buttons

### Disabled accent for dark theme

When something is inactive, don’t use full mint. Use something like: rgba(36, 228, 167, 0.5)

### Sidebar (mandatory) for dark theme

Make the sidebar slightly darker than the main content area in dark mode. Example:

- sidebar: #1b222c
- main background: #202833
